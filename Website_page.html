Website


Objective

To create a brand new website that can showcase my projects so far.

Process

I started by brainstorming what I wanted the site to look like. From the beginning it was clear that I wanted to create a site that was very minimalistic, based around a blue/grey colour palette, with a white background. Originally I made the site using only HTML, CSS with a little bit of Javascript. I had an elaborate plan to have different navigational bars for different page sizes, but this eventually made my CSS confusing and unnecessarily complicated, leading to a bunch of bugs that I couldn't figure out.
After completing the Codecademy course on building websites and learning about Bootstrap, I decided to rewrite the site entirely using Bootstrap's grid. As I wanted the site to not look cluttered yet be a good demonstration of my skill set, I made sure that the different pages kept within the same theme but displayed different elements.
