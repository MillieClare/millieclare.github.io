Objective

To create a functioning bot that sits in my Twitch chat, and responds to commands given to it.

Process

I already had some ideas in mind to create this bot. The bot would interact with chat, using commands that were simple and could be used by anyone, that I had full control over to customise how I wanted. I first did some reading and found out about tmi.js, a library which makes steps to getting the bot to work with the chat much easier. The site itself was also accessible on how to set the bot up to use tmi.js. A useful tool was the twitch developers site, (which can be found HERE) through which I learned about the message handler which the bot uses to decide if an incoming message is a command or not, and if it is a command, what to then do with it.
Currently, the MMB is very simple. She listens to messages and replies with stock messages that have been written for her. My future plans for Miniminibot_ are to introduce more responses for her to give when she recieves a command in order to stop her from sounding too repetitive, a currency for members of the chat to use and minigames for chat to play during the stream (for example, !heist and !challenge minigames from the StreamLabs chat bot are very popular for many streamers).
